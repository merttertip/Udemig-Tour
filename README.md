<h1>Udemig Tour</h1>

<p>
    Bu projede modern bir seyehat acentesi web sitesinin ana sayfasını yansıtan, modüler yapıda inşa edilen interaktif bir React uygulamasıdır.
</p>

<ul>
    <li>Component-Based Architecture: Navbar, Hero, About ve Services gibi alanlar bağımsız bileşenlere ayrılarak tekrar kullanılabilirlik sağlandı.</li>
    <li>Dynamic Rendering: Sosyal medya linkleri ve servis kartları gibi veriler, bir veri dosyasından (data.js) çekilerek dinamik olarak ekrana basıldı.</li>
    <li>Props Usage: Veriler, ana bileşenlerden alt bileşenlere (Title, Service vb.) props yoluyla aktarılarak kod tekrarı önlendi.</li>
</ul>

<p>
   This project is an interactive React application built with a modular structure, reflecting the homepage of a modern travel agency website.
</p>

<ul>
   <li>Component-Based Architecture: Areas such as Navbar, Hero, About, and Services are separated into independent components to ensure reusability.</li>
   <li>Dynamic Rendering: Data such as social media links and service cards are retrieved from a data file (data.js) and dynamically displayed on the screen.</li>
   <li>Props Usage: Data is transferred from main components to sub-components (Title, Service, etc.) via props, preventing code repetition.</li>
</ul>
